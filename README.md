# Softmakers Desafio - Desenvolvedor Full Stack Junior

## **Quickstart**

- Configure o arquivo `.env` com base no arquivo `.env.example`;
- Tenha em sua máquina a versão 16 do node.js. Devido a questões de compatibilidade. Caso preciso de gerenciar várias versões do node.js em sua máquina utilize o [nvm ](https://github.com/nvm-sh/nvm).

### **Rode o servidor**
```bash
npm run dev
```
### **Rode o clinte**
```
npm run react
```
